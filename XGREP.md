## Xgrep utility

This utility makes it easier to use git-grep.

```bash
Usage: xgrep options term(s)
    -d, --[no-]debug
    -f, --file
    -i, --invert
    -p, --include-path PATHS
    -P, --exclude-path PATHS
    -t, --include-type TYPES
    -T, --exclude-type TYPES
    -h, --help
```
